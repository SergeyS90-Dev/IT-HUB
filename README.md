Адаптивный сайт, разработанный с использованием препроцессора Sass.

Основные особенности проекта:
  — Реализована адаптивная верстка (от 360px до 3000px)
  — Использована методология БЭМ для структурирования HTML и CSS
  — Применён Sass (SCSS) для удобной работы со стилями (переменные, вложенность, миксины)
  — Соблюдены базовые принципы доступности (aria-атрибуты, корректная семантика)

Интерактивные элементы:
  — Анимированное логотип (hover-анимация через keyframes)
  — Слайдер в секции architecture (на моюильных разрешениях)
  — Анимации при наведении и взаимодействии с элементами интерфейса

 Оптимизация и производительность:
  — Минифицированы CSS и SVG
  — Оптимизированы изображения
  — Добавлен favicon
  — Настроена приоритетная загрузка ключевых ресурсов (шрифты, изображения)
  — Используется lazy loading для изображений
  — Отсутствуют проблемы с переполнением layout (overflow)
  — Учтены рекомендации Lighthouse

Технологии:
 HTML5, SCSS (Sass), CSS3 (анимации, flex/grid)
